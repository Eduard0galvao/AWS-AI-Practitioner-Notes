# AWS AI Practitioner Notes

Bem-vindo ao meu repositório de estudos para a certificação AWS Certified AI Practitioner (AIF-C01).

Este espaço foi criado para documentar minha jornada de aprendizado em Inteligência Artificial, IA Generativa e serviços de IA da AWS.

Aqui você encontrará:

* Resumos dos conteúdos estudados
* Anotações por domínio da certificação
* Conceitos fundamentais de IA e Machine Learning
* IA Generativa e Modelos de Fundação
* Serviços de IA da AWS
* Engenharia de Prompt
* Questões de prática e simulados
* Materiais de revisão para a prova

## Objetivo

Meu objetivo é consolidar o conhecimento adquirido durante os estudos, criar uma base de consulta para futuras referências e compartilhar aprendizados com a comunidade.

## Estrutura do Repositório

* Domínio 2 - Fundamentos de Modelos de Fundação
